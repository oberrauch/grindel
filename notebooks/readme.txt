
- prepare_idaweb_precip_data.ipynb
	"First contact" with the idaweb data. End result is preprocessed monthly precipitation data from the station Eigergletscher and Kleine Scheidegg between 1908 and 2018.
- get_idabew_data.ipynb
	This notebook covers the whole process of gathering station data from the idaweb service from start to finish. This includes the regional selection, parameter and station selection, preprocessing, outlier correction, ... The processed data is stored in new files.
